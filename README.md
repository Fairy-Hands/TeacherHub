# Teacher Hub

MVP SaaS para professores particulares de inglês.

## Stack

- Next.js
- React
- TypeScript
- TailwindCSS
- Prisma
- PostgreSQL
- NextAuth/Auth.js
- React Hook Form
- Zod
- TanStack Query

## Setup

1. Instale as dependências.
2. Copie `.env.example` para `.env`.
3. Configure `DATABASE_URL`.
4. Rode `prisma generate` e `next dev`.

## Escopo do MVP

- Dashboard
- Cadastro de alunos
- Agenda
- Diário de aula
- Financeiro
